# 🗑️ traSSUre

### Transforming Trash into Treasure with AI

AI-powered Smart Cup Collection & Compression System

---

# 📖 About

**traSSUre** is an AI-powered smart recycling system designed to improve disposable cup recycling efficiency.

The system detects plastic cups using AI, checks for remaining liquid, guides users to empty the cup if necessary, automatically compresses empty cups, and monitors collection status through a web dashboard.

---

# ✨ Features

- 🥤 Plastic cup detection
- 💧 Remaining liquid detection
- ⚠️ "Please empty the cup" guidance
- 🤖 Automatic cup compression
- 📊 Real-time monitoring dashboard
- 📦 Fill-level monitoring
- 🔔 Collection notification
